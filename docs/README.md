## 🔄 ドキュメント更新ガイドライン

### 新規作成時
1. 適切なカテゴリに配置
2. このインデックスに追加
3. ファイル名: `TYPE-簡潔な説明.md`
   - PLAN: 実装計画
   - ADR: アーキテクチャ決定記録
   - GUIDE: 操作ガイド
   - REPORT: 完了レポート

### 更新時
- 変更履歴をファイル末尾に記載
- 古い情報は削除せず「非推奨」として残す
